# 🔐 Ultimate Password Manager (Python Tkinter)

A simple and secure desktop password manager built using Python and Tkinter.
This app helps you generate, store, and retrieve passwords efficiently with a clean GUI.

---

## 🚀 Features

* 🔑 **Password Generator**
  Generates strong, random passwords using letters, numbers, and symbols.

* 💾 **Save Credentials**
  Stores website, email, and password data in a JSON file.

* 🔍 **Search Functionality**
  Quickly retrieve saved credentials for any website.

* 🖥️ **User-Friendly GUI**
  Built with Tkinter for a simple and intuitive interface.

* 🛡️ **Error Handling**
  Handles missing files and invalid inputs gracefully.


---

## 🛠️ Tech Stack

* **Python**
* **Tkinter (GUI)**
* **JSON (Data Storage)**

---

## ⚙️ How to Run

1. Clone the repository:

```
git clone https://github.com/llTheDragonKingll/Ultimate-Password-Manager.git
```

2. Navigate to the project folder:

```
cd Ultimate-Password-Manager
```

3. Run the application:

```
python main.py
```

---

## 📌 Notes

* The app automatically creates a `data.json` file if it doesn't exist.
* A sample file (`data_sample.json`) is included for reference.
